# 寫好廣東話 (Writing Natural Cantonese)

寫出嚟嘅嘢要聽落似人講嘢——自然、清楚、一聽就明。唔好寫到好似教科書或者翻譯機咁生硬。

## 幾時用 (When to Use)

- 寫任何廣東話內容嘅時候
- 審閱或者修改已有嘅廣東話內容
- 將其他語言翻譯做廣東話
- **唔適用：** 刻意寫書面語嘅正式文章

## 正體字原則 (Traditional Character Principles)

**原則：** 簡體轉正體唔係一對一嘅。一個簡體字可以對應幾個唔同嘅正體字，揀錯咗個字意思就唔同晒。要揀啱，睇個字嘅意思，唔好淨係靠自動轉換。

| 簡體 | 正體選項                             |
| ---- | ------------------------------------ |
| 发   | 發（發生）vs 髮（頭髮）              |
| 面   | 面（面部）vs 麵（麵條）              |
| 后   | 后（皇后）vs 後（之後）              |
| 干   | 干（干涉）vs 乾（乾淨）vs 幹（幹活） |
| 里   | 里（鄰里）vs 裏/裡（裏面）           |

**注意：** 有啲簡體字喺正體入面都存在，但意思唔同。例如「干」喺正體入面只係解「干涉」，唔好用嚟表示「乾淨」或「幹活」。

**唔肯定嘅時候：** 諗吓個字喺呢句入面係咩意思，再揀對應嗰個正體字。

## 廣東話詞彙 (Cantonese Vocabulary)

**原則：** 用廣東話人日常會講嘅字。如果一個字讀出嚟覺得怪怪哋，多數就係用咗普通話或者其他地方嘅講法。

| ❌ 唔好用 | ✅ 要用     |
| --------- | ----------- |
| 冰淇淋    | 雪糕        |
| 出租車    | 的士        |
| 洗澡      | 沖涼        |
| 什麼      | 咩 / 乜     |
| 東西      | 嘢          |
| 質量      | 質素 / 質地 |
| 夜宵      | 宵夜        |

**唔肯定嘅時候：** 諗吓你平時同朋友傾偈會唔會用呢個字。唔會嘅話，就搵返廣東話嘅講法。或者成句嘢讀出嚟，如果有個字令你停一停或者覺得唔順口，就換做廣東話嘅講法。

## 異體字與寫法差異 (Character Variations)

**原則：** 廣東話有好多字詞存在多於一種約定俗成嘅寫法。依啲變體發音相近、意思一樣，只係書寫習慣唔同。**唔應該**當佢哋做錯別字，亦唔需要強制統一，揀邊個係作者嘅風格選擇，唔係啱同錯嘅問題。

| 常用寫法 | 可接受變體 |
| -------- | ---------- |
| 依啲     | 呢啲       |
| 咁       | 噉         |
| 嘅       | 既         |
| 裏       | 裡         |
| 而家     | 宜家       |

**注意：** 如果作者已經喺文中用開某個寫法，可以建議保持一致，但呢個只係風格建議，唔係改錯。

## 中英夾雜 (Code-Mixing)

**原則：** 香港人講嘢寫嘢好自然會夾雜英文。特別係科技用語、工作日常或者習以為常嘅外來語，直接用英文會比起生硬翻譯自然得多。

| ❌ 唔好生硬翻譯 | ✅ 直接用英文或夾雜 |
| --------------- | ------------------- |
| 開啟文件        | 開個 file           |
| 點擊連結        | 撳條 link           |
| 應用程式        | App                 |

**注意：** 唔好為夾雜而夾雜。如果個字本身有極度普遍嘅廣東話對應（例如將「好危險」寫成「好 dangerous」），夾硬用英文反而會極度唔自然。只有喺「如果翻譯成中文會好生硬」或者「大家平時都習慣講英文」嘅前提下，先應該中英夾雜。

## 句式同語法 (Sentence Patterns & Grammar)

### 書面語句式要避開 (Written-Style Patterns to Avoid)

**原則：** 廣東話口語嘅句式同書面語好唔同。如果句嘢讀出嚟好似喺度讀緊文章，就要改做口語講法。

| ❌ 書面語     | ✅ 廣東話             |
| ------------- | --------------------- |
| 因為 A 所以 B | A 所以 B ／ A 咪 B 囉 |
| 雖然 A 但是 B | A 不過 B ／ A 但係 B  |
| 如果 A 就 B   | A 嘅話就 B            |
| 非常好        | 好正 ／ 勁正          |
| 他／她        | 佢                    |

### 語氣助詞 (Sentence-Final Particles)

**原則：** 語氣助詞令句嘢聽落自然，但用錯咗會好怪。**寧願唔加都唔好亂加**——冇語氣詞嘅句子係正常嘅，用錯語氣詞嘅句子會好突兀。

| 助詞 | 語氣         | 例子         |
| ---- | ------------ | ------------ |
| 㗎   | 肯定、解釋   | 係咁㗎。     |
| 啦   | 提議、催促   | 走啦！       |
| 喎   | 發現、提醒   | 原來係咁喎。 |
| 囉   | 理所當然     | 咪係囉。     |
| 咩   | 反問         | 你唔知咩？   |
| 喇   | 完成、變化   | 夠喇。       |
| 啩   | 推測、唔肯定 | 應該係啩。   |

### 廣東話特有句式 (Cantonese-Specific Constructions)

- **完成式問句：** 食咗飯未？（唔好寫「你吃過飯了嗎？」）
- **方向補語：** 行過去、揸返嚟、搬上去
- **後置修飾：** 食埋佢、做先、買多啲添

## 粗口與地道助語詞 (Profanity & Slang)

**原則：** 視乎語境，例如需要強烈情緒表達、搞笑、或者極之 casual 嘅對話語境，適當地加入廣東話粗口（例如「屌」、「鳩」、「撚」、「柒」、「閪」，或者相關諧音字）可以令表達更傳神同有力度。

| ❌ 生硬或者平淡 | ✅ 傳神粗口 / 俗語 | 語法說明                    |
| --------------- | ------------------ | --------------------------- |
| 黐線            | 痴撚線 / 痴鳩線    | 作為中綴 (Infix) 加強形容詞 |
| 好慘啊          | 仆街喇             | 地道俗語直接代替形容詞      |
| 忘記帶鎖匙      | 屌！唔記得帶鎖匙   | 句首直接作為強烈感嘆詞      |

**注意：** 唔好為加而加。粗口嘅擺位必須符合廣東話獨特嘅文法（如上表嘅「中綴」結構）。

## 常見錯誤 (Common Mistakes)

| 錯誤                     | 修正                                 |
| ------------------------ | ------------------------------------ |
| 用書面語寫廣東話         | 讀出嚟，唔順口就改                   |
| 揀錯正體字               | 按意思揀字，唔好靠自動轉換           |
| 用普通話詞彙             | 換做廣東話日常講嘅字                 |
| 亂加語氣助詞             | 唔肯定就唔加，好過用錯               |
| 句式太書面               | 用口語句式，避開因為…所以…等寫法     |
| 正體字正確但語感似書面語 | 單字啱唔代表成句啱，要成句讀出嚟聽吓 |
