## 正體字對照表 (Traditional Character Lookup)

以下係常見嘅簡體轉正體容易揀錯嘅字。留意有啲字（例如「干」、「后」、「里」）喺正體本身已經存在，但意思同簡體完全唔同（即 False Friends 面對面陷阱）。揀嘅時候一定要睇個字喺句子入面嘅意思。

| 簡體 | 正體選項       | 正確用法                                               |
| ---- | -------------- | ------------------------------------------------------ |
| 发   | 發 vs 髮       | 發生、發展、發現（發）；頭髮、理髮（髮）               |
| 面   | 面 vs 麵       | 面前、面部、見面（面）；食麵、麵包（麵）               |
| 后   | 后 vs 後       | 皇后、天后（后）；之後、後面、最後（後）               |
| 干   | 干 vs 乾 vs 幹 | 干涉、干擾（干）；乾淨、乾燥（乾）；幹嘛、幹活（幹）   |
| 里   | 里 vs 裏/裡    | 鄰里、公里（里）；裏面、裡邊、心裏（裏/裡）            |
| 冲   | 沖 vs 衝       | 沖涼、沖茶、沖水（沖）；衝動、衝出去、衝突（衝）       |

| 松   | 松 vs 鬆       | 松樹、松鼠（松）；放鬆、鬆手、鬆散（鬆）               |
| 钟   | 鍾 vs 鐘       | 鍾意、鍾情（鍾）；時鐘、鐘頭、鐘樓（鐘）               |
| 台   | 台 vs 臺 vs 檯 | 台灣（台/臺）；寫字檯、檯面（檯）；電台（台）          |

| 坛   | 壇 vs 罈       | 論壇、花壇（壇）；酒罈、醬罈（罈）                     |

| 系   | 系 vs 係 vs 繫 | 系統、中文系（系）；係咪、唔係（係）；繫好安全帶（繫） |
| 几   | 几 vs 幾       | 茶几（几）；幾多、幾時（幾）                           |

## 語氣詞詳細用法 (Particle Reference)

### 㗎 (gaa3/gaa4)

**語氣：** 肯定、解釋、強調事實

- 係咁㗎。（就係咁嘅。）
- 我一直都係咁做㗎。（我向來都係咁做。）
- 升調「㗎？」變成疑問：真㗎？（真的嗎？）

### 啦 (laa1)

**語氣：** 提議、催促、輕微命令

- 走啦！（快走吧！）
- 食飯啦。（吃飯了。）
- 算啦。（算了吧。）

### 喎 (wo3/wo5)

**語氣：** 發現新事物、提醒對方注意

- 原來係咁喎。（原來是這樣啊。）
- 佢好叻喎。（他很厲害哦。）
- 今日落雨喎。（今天下雨了哦。）

### 囉 (lo1)

**語氣：** 理所當然、顯而易見

- 咪係囉。（可不是嘛。）
- 你自己諗囉。（你自己想吧。）
- 去咪去囉。（去就去呗。）

### 咩 (me1)

**語氣：** 反問、驚訝、質疑

- 你唔知咩？（你不知道嗎？）
- 係咁咩？（是這樣嗎？）
- 有咁嘅事咩？（有這種事嗎？）

### 喇 (laa3)

**語氣：** 完成、狀態變化

- 夠喇。（夠了。）
- 我走喇。（我走了。）
- 搞掂喇。（搞定了。）

### 啩 (gwaa3)

**語氣：** 推測、不太肯定

- 應該係啩。（應該是吧。）
- 佢會嚟啩。（他會來吧。）
- 冇問題啩。（沒問題吧。）

### 吖 (aa1)

**語氣：** 友善提問、柔化語氣

- 你幾時得閒吖？（你什麼時候有空呀？）
- 食咩吖？（吃什麼呀？）

### 㗎嘛 (gaa3 maa3)

**語氣：** 理所當然、事情本來就係咁

- 人哋都係咁做㗎嘛。（人家都是這樣做的嘛。）

### 嘅 (ge3)

**語氣：** 所有格 ／ 肯定語氣

- 我嘅書。（我的書。）
- 會搞掂嘅。（會搞定的。）

### 幾時唔加語氣詞 (When to Omit Particles)

**原則：** 冇語氣詞嘅句子係完全正常嘅。唔好為咗「聽落似廣東話」就每句都加語氣詞。

以下情況唔使加：

- 陳述事實：「佢琴日返咗嚟。」（唔使加㗎、喇）
- 列舉：「材料有雞蛋、麵粉、糖。」
- 說明步驟：「首先開個 file，然後寫入內容。」

**唔肯定加邊個就唔好加。** 用錯語氣詞比冇語氣詞更加唔自然。

## 粗口與傳神助語詞 (Profanity & Slang Usage)

廣東話粗口有獨特嘅語法規則，用得準確可以精妙地表達情緒。注意：要根據場合決定用正字定係諧音/委婉語。

### 常見粗字與用法

| 粗字 / 俗字 | 意思 / 用法                     | 例子                                       | 委婉語 / 諧音 |
| ----------- | ------------------------------- | ------------------------------------------ | ------------- |
| 屌          | 表達強烈不滿，或者做動詞        | 「屌！又唔記得帶鎖匙。」、「畀人屌到飛起」 | 頂、妖、丟    |
| 鳩          | 傻、多餘、強調程度 (通常帶負面) | 「好鳩煩」、「戇鳩」                       | 膠、狗        |
| 撚          | 強調語氣 (常做副詞)，或指人     | 「食撚咗未？」、「搞撚錯」、「痴撚線」     | 鬼、諗        |
| 柒          | 羞恥、肉酸、笨拙                | 「好柒呀」、「柒咗」                       | 賊、七        |
| 閪          | 惡劣、倒霉、差勁                | 「好閪衰」、「閪面」                       | 西、鞋        |

### 獨特語法：粗口中綴 (Infix)

廣東話粗口最強大嘅地方係可以當「中綴」插入詞語中間，加強語氣。

- **形容詞：** 痴線 ➡️ 痴 **撚** 線 ／ 離譜 ➡️ 離 **鳩** 譜
- **動詞短語：** 搞錯 ➡️ 搞 **撚** 錯 ／ 走呀 ➡️ 走 **撚** 呀

### 地道俗語 (Slang)

- **仆街 (Puk Gaai)：** 可以做動詞 (跌倒)、名詞 (壞人)、形容詞 (慘了) ➡️ 「今次仆街喇！」
- **頂你個肺：** 表達驚訝或不滿。
- **黐線 / 痴線：** 瘋狂，多數配搭粗口使用。
